Existing AI Agent

Modify Existing AI Agent

Add features or improve behavior

1 Go to the Project Folder
cd ~/Desktop/Projects/aiagents/existing-agent
2 Check Git Status
git status && git branch
3 Start Claude Code
claude
4 Explore the Agent Code
/brownfield-explore

Tell Claude:

"Show me how this agent works. Where is the system prompt? What tools/functions does it have? How does it save conversations?"
5 Create Feature Branch
git checkout -b feature/add-calendar-tool
6 Make Your Changes

To change agent behavior:

/gsd:quick
"Update the system prompt so the agent is more concise and always asks clarifying questions before giving long answers."

To add a new tool:

/gsd:quick
"Add a new tool called 'check_calendar' that lets the agent look up available meeting slots. It should query the Supabase 'calendar_slots' table."

To add a new integration:

/channel-integration
"Add Slack integration so this agent can respond to messages in Slack channels."
7 Test the Agent

Have test conversations:

  • Test the new feature works
  • Test old features still work
  • Test edge cases (weird inputs)
  • Test same scenario multiple times
8 If Agent Misbehaves
/sf
"The agent isn't using the new calendar tool. It just says 'I can't check calendars' even though I added the tool."
9 Save & Deploy
/git-flow /quick-deploy

Before You Deploy - Check These

  • Explored code first
  • New feature works
  • Old features work
  • Tested multiple times
  • No API keys exposed
  • Error handling works
