Build a New Website

Next.js + Supabase + Vercel

1 Open Terminal & Go to Project Folder
cd ~/Desktop/Projects/websites mkdir my-new-site cd my-new-site
2 Start Claude Code
claude
3 Initialize Your Project

Type this command, then answer Claude's questions about what you're building.

/gsd:new-project

Example answers:

"I'm building a landing page for a dentist clinic. It needs a hero section, services section, about section, contact form, and testimonials. Use Next.js 15, Supabase for the contact form submissions, and deploy to Vercel."
4 Discuss Phase 1

Claude will ask questions to understand exactly what to build.

/gsd:discuss-phase 1
5 Plan Phase 1
/gsd:plan-phase 1
6 Build Phase 1
/gsd:execute-phase 1
7 Repeat for Each Phase

Keep going through all phases in your roadmap:

/gsd:discuss-phase 2 /gsd:plan-phase 2 /gsd:execute-phase 2

Then phase 3, 4, etc. until done.

8 If You Need Nice UI Design
/fd

Then tell Claude what you need:

"Design a modern hero section with a gradient background, large headline, subtext, and a CTA button"
9 If You Need Database Tables
/sb

Then tell Claude what you need:

"Create a contact_submissions table with name, email, phone, message, and created_at columns. Add RLS policy so only authenticated users can read."
10 Check Everything Works
/gsd:verify-work
11 Save Your Code
/git-flow
12 Deploy to Vercel
/quick-deploy
13 Mark Project Complete
/gsd:complete-milestone

Before You Deploy - Check These

  • Site works on mobile
  • All links work
  • Forms submit correctly
  • Images load fast
  • No console errors
  • Supabase env vars in Vercel
