  <action>
Add button micro-interactions to app/globals.css using data attribute selectors (not class-based — prevents Tailwind conflicts):

```css
@layer base {
  /* Button micro-interactions */
  [data-slot='button'] {
    @apply transition-all duration-200;
  }

  /* Hover: lift effect */
  [data-slot='button']:not(:disabled):hover {
    @apply -translate-y-[1px];
  }

  /* Active: press down effect */
  [data-slot='button']:not(:disabled):active {
    @apply translate-y-0 scale-[0.98];
  }
}
```

- Ensure data-slot="button" is present (already on line 53)
