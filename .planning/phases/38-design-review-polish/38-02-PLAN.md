  <action>
Read the 38-01 audit report (`38-01-AUDIT.md`) for dashboard and projects findings.

**Fix all CRITICAL and HIGH violations. Common fixes include:**

1. **Replace pure grays with tinted neutrals:**
   - `gray-*` classes -> semantic tokens (`text-muted-foreground`, `bg-muted`, `border-border`, `bg-secondary`)
   - `text-black` -> `text-foreground`
   - `bg-white` -> `bg-card` or `bg-background`
   - `#000000`, `#ffffff` -> CSS variable equivalents

2. **Replace arbitrary shadows with elevation system:**
   - `shadow-sm` -> `shadow-elevation-1`
   - `shadow-md` -> `shadow-elevation-2`
   - `shadow-lg` -> `shadow-elevation-3`
   - `shadow-xl` -> `shadow-elevation-4`

3. **Fix surface layering:**
   - Ensure three-tier depth: `bg-background` (base) > `bg-card` / `hsl(var(--surface-1))` (cards) > `hsl(var(--surface-2))` (nested)
   - No cards wrapping cards at same elevation level

4. **Fix spacing inconsistencies:**
   - Replace random `p-3 gap-2 mt-5` patterns with consistent spacing scale
   - Use fluid spacing where applicable: `var(--space-sm)`, `var(--space-md)`, `var(--space-lg)`

5. **Fix motion timing:**
   - Replace `ease-in-out`, `linear`, `duration-300` with `ease-out-quart` / `ease-premium` and `duration-200`/`duration-[250ms]`

Apply fixes per the specific audit findings. Do NOT change functionality — only styling/classes.

After fixes, run `npx tsc --noEmit` to ensure no type errors introduced.
</action>
