   - Return true if link exists, false otherwise
   - Used to verify client has access to specific project

3. **getClientProjectIds(userId: string): Promise&lt;string[]&gt;**
   - Query client_projects table for all projects where client_id = userId
   - Return array of project IDs
   - Used for listing client's projects

Pattern:

```typescript
import { createClient } from '@/lib/supabase/server';

export async function isClientRole(userId: string): Promise<boolean> {
  const supabase = await createClient();
  const { data } = await supabase.from('profiles').select('role').eq('id', userId).single();
  return data?.role === 'client';
}
```

Follow similar pattern for other functions.
