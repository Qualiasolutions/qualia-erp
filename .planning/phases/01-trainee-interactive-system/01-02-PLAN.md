<task type="auto">
  <name>Create project-integrations server actions</name>
  <files>app/actions/project-integrations.ts</files>
  <action>
Create server actions for managing project_integrations table:

1. getProjectIntegrations(projectId: string):
   - Query project_integrations WHERE project_id = projectId
   - ORDER BY service_type ASC
   - Return array of { id, service_type, external_url, external_id, metadata }
   - Return empty array if no integrations found

2. upsertIntegration(projectId: string, serviceType: string, externalUrl: string):
   - Validate inputs (Zod schema: url must be valid URL, serviceType in ['github', 'vercel'])
   - Check if integration exists for this project + serviceType
   - If exists: UPDATE external_url WHERE project_id AND service_type
   - If not exists: INSERT new row with project_id, service_type, external_url, connected_at = now()
   - revalidatePath(`/projects/${projectId}`)
   - Return ActionResult

3. deleteIntegration(integrationId: string, projectId: string):
   - DELETE FROM project_integrations WHERE id = integrationId
   - revalidatePath(`/projects/${projectId}`)
   - Return ActionResult

All actions use createClient from @/lib/supabase/server and follow ActionResult pattern.
</action>
