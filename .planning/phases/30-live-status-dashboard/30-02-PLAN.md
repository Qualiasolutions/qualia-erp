<task type="auto">
  <name>Task 1: Create LiveStatusPanel component</name>
  <files>components/today-dashboard/live-status-panel.tsx</files>
  <action>
Create `components/today-dashboard/live-status-panel.tsx` as a 'use client' component.

**Props:** `workspaceId: string`

**Data:** Use `useTeamStatus(workspaceId)` from `lib/swr.ts`.

**Layout:** A compact panel/card that fits alongside existing dashboard widgets. Use the same card style as other dashboard sections (check `inbox-widget.tsx` or `meetings-sidebar.tsx` for pattern).

**For each team member, render a row:**

- Avatar (use shadcn Avatar with initials fallback from full_name)
- Full name
- Status indicator:
  - **Online (status === 'online'):** Green dot (pulsing subtle animation via CSS, NOT framer-motion). Show project name. Show live ticking duration since `sessionStartedAt` — use a `useEffect` with `setInterval(1000)` to update a local "elapsed" state every second. Format as "Xh Ym" or "Xm" if under 1 hour.
  - **Offline (status === 'offline'):** Grey dot. Show "Last seen X ago" using `formatDistanceToNow` from `date-fns`. If `lastSessionEndedAt` is null, show "No sessions yet".

**Styling:**

- Brand tinted neutrals, not pure grey/black
- Green indicator: use `bg-emerald-500` with a subtle pulse (`animate-pulse` but slowed down via custom CSS or Tailwind `animate-[pulse_3s_ease-in-out_infinite]`)
- Grey indicator: `bg-muted-foreground/40`
- Clean, data-dense layout like Linear — no decorative elements
- Show a small header: "Team Status" with a subtle live indicator dot
- Loading state: skeleton rows (use shadcn Skeleton)
- Empty state: "No team members found"

**Duration ticker implementation:**

```typescript
// Inside each online member row, use local state:
const [elapsed, setElapsed] = useState(() => calcElapsed(sessionStartedAt));
useEffect(() => {
  const interval = setInterval(() => setElapsed(calcElapsed(sessionStartedAt)), 1000);
  return () => clearInterval(interval);
}, [sessionStartedAt]);
```

Extract the ticker into a small `DurationTicker` sub-component to avoid re-rendering the entire list every second. Use `React.memo` on the member row component.
</action>
