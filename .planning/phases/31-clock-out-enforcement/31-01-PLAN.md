1. Create `hooks/use-idle-detection.ts`:
   - Export `useIdleDetection(options: { enabled: boolean; idleTimeout: number; graceTimeout: number; onIdle: () => void; onAutoClose: () => void })`.
   - `idleTimeout` default: 30 * 60 * 1000 (30 min). `graceTimeout` default: 15 * 60 * 1000 (15 min).
   - Track `mousemove`, `keydown`, `scroll`, `touchstart`, `click` events on `document`.
   - Use a single `setTimeout` approach: after `idleTimeout` ms of no activity, call `onIdle`. Start a second timer for `graceTimeout`. If no activity during grace, call `onAutoClose`.
   - Return `{ isIdle: boolean; isGracePeriod: boolean; resetIdle: () => void }`.
   - When `enabled` is false, clear all timers and remove all listeners. Use `useEffect` cleanup.
   - Throttle event handlers to avoid performance issues â€” use a ref to track last activity time and only reset timer if >5s since last reset.
   - Add `'use client'` directive.

2. Add `autoClockOut` to `app/actions/work-sessions.ts`:
   - Export `async function autoClockOut(workspaceId: string, sessionId: string, reason: string): Promise<ActionResult>`.
   - Similar to `clockOut` but does NOT require a user-written summary â€” uses the `reason` parameter (e.g., "[Auto-closed: inactivity]").
   - Validates session belongs to current user, session is still open.
   - Sets `ended_at` to now, `summary` to the reason string.
   - Calls `revalidatePath('/')`.
