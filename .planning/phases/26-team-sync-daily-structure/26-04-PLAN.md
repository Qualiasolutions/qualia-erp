    ```typescript
    export async function createDailyCheckin(
      workspaceId: string,
      whatDid: string,
      whatBlocked: string | null
    ): Promise<ActionResult>
    ```
    - Auth: get user from `supabase.auth.getUser()`, return 'Unauthorized' if missing
    - Insert into `daily_checkins` with `profile_id = user.id`, `workspace_id`, `checkin_date = CURRENT_DATE` (send as `new Date().toISOString().split('T')[0]`)
    - On conflict (profile_id, checkin_date): upsert (update `what_did`, `what_blocked`, `updated_at`)
    - `revalidatePath('/')`
    - Return `ActionResult`

    ```typescript
    export async function getTodaysCheckin(workspaceId: string): Promise<ActionResult<DailyCheckin | null>>
    ```
    - Get current user, query `daily_checkins` where `profile_id = user.id` AND `checkin_date = today`, `.maybeSingle()`
    - Return data (may be null — user hasn't checked in yet)

    ```typescript
    export async function getCheckins(
      workspaceId: string,
      options: { profileId?: string; date?: string; limit?: number }
    ): Promise<ActionResult<DailyCheckin[]>>
    ```
    - Admin only (check `profiles.role = 'admin'`). Return 403 if not admin.
    - Query `daily_checkins` joined with `profile:profiles(id, full_name, avatar_url)`
    - Apply filters: `profileId` → `.eq('profile_id', profileId)`, `date` → `.eq('checkin_date', date)`
    - Order by `created_at DESC`, limit to options.limit || 50

    Export `DailyCheckin` type matching the DB row + profile join.

    **Create `app/actions/time-logs.ts`** (`'use server'` directive):

    ```typescript
    export async function startTaskTimer(taskId: string): Promise<ActionResult>
    ```
    - Auth check
    - Upsert into `task_time_logs`: `{ task_id: taskId, profile_id: user.id, started_at: now(), finished_at: null }`
    - On conflict (task_id, profile_id): update `started_at = now()`, clear `finished_at`

    ```typescript
    export async function stopTaskTimer(taskId: string): Promise<ActionResult<{ minutes: number }>>
    ```
    - Auth check
    - Fetch existing `task_time_logs` row for this task + user
    - If no row or `started_at` is null: return error 'Timer not started'
    - Update `finished_at = now()` — the `time_spent_minutes` generated column computes automatically
    - Re-fetch the row to get `time_spent_minutes`
    - Return `{ success: true, data: { minutes: row.time_spent_minutes } }`

    ```typescript
    export async function getTaskTimeLog(taskId: string): Promise<ActionResult<TaskTimeLog | null>>
    ```
    - Fetch `task_time_logs` for `task_id` and current user, `.maybeSingle()`

    ```typescript
    export async function getTaskTimeLogs(taskIds: string[]): Promise<ActionResult<TaskTimeLog[]>>
    ```
    - Admin only. Fetch all logs for an array of task IDs with profile join. Used for team dashboard.

    Export `TaskTimeLog` type.

    **In `app/actions/inbox.ts` — wire task completion notification:**

    In `quickUpdateTask`, after the successful `.update()` when `updates.status === 'Done'`, add (fire-and-forget):

    ```typescript
    if (updates.status === 'Done') {
      // Get task details for notification
      const { data: completedTask } = await supabase
        .from('tasks')
        .select('title, workspace_id, creator_id, assignee_id')
        .eq('id', taskId)
        .single();

      if (completedTask) {
        // Get admin profiles to notify
        const { data: admins } = await supabase
          .from('profiles')
          .select('id')
          .eq('role', 'admin')
          .neq('id', user.id); // Don't notify yourself

        const { createNotification } = await import('@/app/actions/notifications');

        for (const admin of admins || []) {
          createNotification(
            admin.id,
            completedTask.workspace_id,
            'task_completed',
            'Task completed',
            `${(await supabase.from('profiles').select('full_name').eq('id', user.id).single()).data?.full_name || 'Someone'} completed "${completedTask.title}"`,
            `/`,
            { task_id: taskId }
          ).catch((err) => console.error('[quickUpdateTask] Notification error:', err));
        }
      }
    }
    ```

    Note: Import `createNotification` dynamically to avoid circular import issues. Place this block after the DB update succeeds and before `revalidatePath`.
