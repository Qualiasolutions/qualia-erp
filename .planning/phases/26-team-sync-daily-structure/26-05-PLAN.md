  <action>
    Create `app/actions/team-dashboard.ts` (`'use server'` directive):

    ```typescript
    export type TeamMemberTasks = {
      profile: {
        id: string;
        full_name: string | null;
        avatar_url: string | null;
        role: string;
      };
      tasks: {
        id: string;
        title: string;
        status: 'Todo' | 'In Progress' | 'Done';
        priority: string;
        due_date: string | null;
        assignee_id: string;
        project: { id: string; name: string } | null;
        time_log: { started_at: string | null; finished_at: string | null; time_spent_minutes: number | null } | null;
      }[];
    };
    ```

    ```typescript
    export async function getTeamTaskDashboard(workspaceId: string): Promise<ActionResult<TeamMemberTasks[]>>
    ```

    Implementation:
    1. Auth check — get `user.id`
    2. Get current user's role from profiles
    3. If `role === 'admin'`: fetch all employee profiles in workspace (`role = 'employee'`), then fetch their tasks
    4. If `role === 'employee'`: only return tasks for `user.id` (single-member result)
    5. For each profile, fetch tasks — include `assignee_id` in the select:
       ```
       tasks table: select id, title, status, priority, due_date, assignee_id, project_id
       WHERE assignee_id = profile.id AND workspace_id = workspaceId AND status IN ('Todo', 'In Progress')
       + left join task_time_logs on task_id = tasks.id AND profile_id = profile.id (select started_at, finished_at, time_spent_minutes)
       + left join projects on project_id = projects.id (select id, name)
       ```
    6. Group into `TeamMemberTasks[]` array sorted by: employee name alphabetically
    7. Within each member: sort tasks by priority (Urgent > High > Medium > Low > No Priority), then due_date ASC

    Use `Promise.all` to fetch all employees' tasks in parallel.

    Export a `useTeamTaskDashboard` SWR hook in `lib/swr.ts`:
    ```typescript
    export function useTeamTaskDashboard(workspaceId: string) {
      return useSWR(
        workspaceId ? cacheKeys.teamDashboard(workspaceId) : null,
        () => getTeamTaskDashboard(workspaceId),
        { ...swrConfig, refreshInterval: isDocumentVisible() ? 45000 : 0 }
      );
    }
    ```
    Add `teamDashboard: (workspaceId: string) => \`team-dashboard-${workspaceId}\`` to cacheKeys.
