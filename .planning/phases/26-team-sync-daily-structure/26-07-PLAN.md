    **Create `components/task-time-tracker.tsx`** (`'use client'`):

    Props:
    ```typescript
    interface TaskTimeTrackerProps {
      taskId: string;
      initialTimeLog: {
        started_at: string | null;
        finished_at: string | null;
        time_spent_minutes: number | null;
      } | null;
      onUpdate?: () => void;
    }
    ```

    Implementation:
    - Three states: "not started", "running" (has started_at, no finished_at), "done" (has both)
    - "not started": show a `Play` icon button (`lucide-react Play` icon, `size-3.5`), tooltip "Start timer"
    - "running": show elapsed time counter (live, updates every 10s via `useEffect` + `setInterval`) + `Square` (stop) icon button
    - "done": show total time formatted ("42m", "1h 12m") in a muted text badge — no button (timer already stopped)
    - On Play click: call `startTaskTimer(taskId)`, optimistically switch to running state
    - On Stop click: call `stopTaskTimer(taskId)`, get returned minutes, switch to done state
    - Button styling: ghost, icon-only, `size="icon" className="size-6"` — minimal, doesn't dominate the card
    - Elapsed counter while running: calculate `now - started_at` in seconds, format as "Xm" or "Xh Xm"

    **In `components/today-dashboard/team-task-card.tsx`**:

    Update `TeamTaskCard` to accept additional props for time tracking:

    ```typescript
    interface TeamTaskCardProps {
      task: {
        id: string;
        title: string;
        status: 'Todo' | 'In Progress' | 'Done';
        priority: string;
        due_date: string | null;
        assignee_id: string;   // snake_case — from TeamMemberTasks type
        project: { id: string; name: string } | null;
        time_log: { started_at: string | null; finished_at: string | null; time_spent_minutes: number | null } | null;
      };
      currentUserId: string;
      onTaskUpdate?: () => void;
    }
    ```

    At end of card row, after due date chip, add the time tracker:
    ```tsx
    {currentUserId === task.assignee_id && (
      <TaskTimeTracker
        taskId={task.id}
        initialTimeLog={task.time_log}
        onUpdate={onTaskUpdate}
      />
    )}
    ```

    Note: use `task.assignee_id` (snake_case) — NOT `task.assigneeId`. The `TeamMemberTasks` type from plan 26-05 uses snake_case throughout.

    - Only show the time tracker if the current user is the assignee (employees track their own tasks)
    - Admins see `time_spent_minutes` as read-only text badge (already in the card design), not the interactive tracker

    **In `components/today-dashboard/team-task-container.tsx`**:
    - Update `TeamTaskCard` usage to pass `currentUserId` and `onTaskUpdate` props to each card.
    - `onTaskUpdate` can call the SWR cache invalidation for `teamDashboard`.

    Run `npx tsc --noEmit` and fix any type errors.
