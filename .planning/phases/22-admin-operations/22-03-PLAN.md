  <name>Task 1: Add resetClientPassword server action</name>
  <files>app/actions/client-portal.ts</files>
  <action>
    Add `resetClientPassword(email: string): Promise<ActionResult>` exported function.

    Implementation:
    1. Auth + admin check (isUserManagerOrAbove)
    2. Normalize email to lowercase
    3. Validate email is non-empty string
    4. Generate new temp password: `const tempPassword = 'Qualia-' + randomBytes(6).toString('hex') + '!';`
       (node:crypto randomBytes already imported — same pattern as setupPortalForClient)
    5. Look up user in auth by email using admin client:
       ```typescript
       const adminClient = createAdminClient();
       const { data: listData } = await adminClient.auth.admin.listUsers({ perPage: 1000 });
       const authUser = listData?.users?.find(u => u.email?.toLowerCase() === email);
       ```
       If not found: return `{ success: false, error: 'No portal account found for this email' }`
    6. Update password:
       ```typescript
       await adminClient.auth.admin.updateUserById(authUser.id, { password: tempPassword });
       ```
       If error: return `{ success: false, error: updateError.message }`
    7. Return: `{ success: true, data: { email, tempPassword, name: authUser.user_metadata?.full_name || null } }`

    Note: This does NOT send an email — it returns the password for Moayad to share manually, which matches the existing pattern for new accounts.

  </action>
