    **Cron auth hardening (all 7 files):**

    In each cron route, remove the `if (process.env.NODE_ENV === 'production')` wrapper around the auth check. The auth check must run unconditionally in ALL environments. Replace the pattern:

    ```typescript
    // OLD — only checks in production
    if (process.env.NODE_ENV === 'production') {
      if (!cronSecret || authHeader !== `Bearer ${cronSecret}`) {
        return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
      }
    }
    ```

    With:

    ```typescript
    // NEW — always checks (returns 401 if CRON_SECRET not set OR doesn't match)
    if (!cronSecret || authHeader !== `Bearer ${cronSecret}`) {
      console.error('[cron/ROUTE_NAME] Unauthorized request');
      return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
    }
    ```

    The 7 cron routes are:

    - `app/api/cron/uptime-check/route.ts`

    - `app/api/cron/weekly-digest/route.ts`
