**WorkSession type** (inferred from Tables<'work_sessions'> + joined profile/project):

```ts
export interface WorkSession {
  id: string;
  workspace_id: string;
  profile_id: string;
  project_id: string | null;
```

**clockIn(workspaceId: string, projectId: string): Promise<ActionResult>**

- Auth: get user via supabase.auth.getUser(), return error if not authenticated
- Check for existing open session: query work_sessions where profile_id = user.id AND workspace_id = workspaceId AND ended_at IS NULL. If found, return { success: false, error: 'You already have an active session. Clock out first.' }
- Insert new row: { workspace_id, profile_id: user.id, project_id: projectId, started_at: new Date().toISOString() }
- After insert: call revalidatePath('/')
- Return { success: true, data: insertedRow }

**clockOut(workspaceId: string, sessionId: string, summary: string): Promise<ActionResult>**

- Auth: get user via supabase.auth.getUser()
- Validate: summary.trim() must be non-empty, else return { success: false, error: 'Summary is required.' }
- Fetch the session: work_sessions where id = sessionId AND profile_id = user.id AND workspace_id = workspaceId AND ended_at IS NULL. If not found return { success: false, error: 'No active session found.' }
- Compute duration_minutes: Math.round((Date.now() - new Date(session.started_at).getTime()) / 60000)
- Update the row: { ended_at: new Date().toISOString(), summary: summary.trim(), duration_minutes }
- After update: revalidatePath('/')
- Return { success: true, data: updatedRow }

**getActiveSession(workspaceId: string): Promise<WorkSession | null>**

- Auth: get user, return null if not authenticated
- Query work_sessions with select `*, project:projects!work_sessions_project_id_fkey (id, name)` where profile_id = user.id AND workspace_id = workspaceId AND ended_at IS NULL, limit 1
- Normalize project FK (Array.isArray check, take [0])
- Return row or null

**getTodaysSessions(workspaceId: string): Promise<WorkSession[]>**

- Auth: get user, return [] if not authenticated
- today = new Date().toISOString().split('T')[0]
- Query work_sessions with select `*, project:projects!work_sessions_project_id_fkey (id, name)` where profile_id = user.id AND workspace_id = workspaceId AND started_at >= today + 'T00:00:00.000Z' (use gte filter)
- Order by started_at ascending
- Normalize project FK on each row
- Return rows

**getSessionsAdmin(workspaceId: string, options: { profileId?: string; date?: string; limit?: number } = {}): Promise<WorkSession[]>**

- Auth: get user, check isUserAdmin (import from ./shared). Return [] if not admin.
- Build query: work_sessions with select `*, profile:profiles!work_sessions_profile_id_fkey (id, full_name, avatar_url), project:projects!work_sessions_project_id_fkey (id, name)` where workspace_id = workspaceId
- Apply optional filters: profileId → eq('profile_id', profileId), date → use gte/lte on started_at for that calendar day (gte date+'T00:00:00Z' AND lt next day+'T00:00:00Z'), limit → limit(options.limit)
- Order by started_at descending
- Normalize profile and project FKs on each row
- Return rows

Import pattern at top: `import { createClient } from '@/lib/supabase/server'`, `import { revalidatePath } from 'next/cache'`, `import { isUserAdmin } from './shared'`, `import type { ActionResult } from './shared'`
