**Behavior:**

- `'use client'`
- On mount (when open becomes true AND currentUserId is not null), fetch assigned projects via `getEmployeeAssignments(currentUserId)`.

**CRITICAL — getEmployeeAssignments returns ActionResult, not an array.** Unwrap correctly:

```ts
const result = await getEmployeeAssignments(currentUserId);
const assignments =
  (result.data as Array<{ project: { id: string; name: string; status: string } }>) ?? [];
const activeProjects = assignments
  .filter((a) => a.project?.status === 'Active')
  .map((a) => ({ id: a.project.id, name: a.project.name }))
  .sort((a, b) => a.name.localeCompare(b.name));
```

Do NOT treat `result` as an array directly — it is `{ success: boolean, data?: unknown }`.

- State: `selectedProjectId: string`, `projects: {id: string; name: string}[]`, `loadingProjects: boolean`, `error: string | null`, `isPending: boolean` (for submission)
- On submit: call `clockIn(workspaceId, selectedProjectId)`. On success: `invalidateActiveSession(workspaceId, true)`, then call `onSuccess()`.
- The Dialog must be undismissable: `onEscapeKeyDown={(e) => e.preventDefault()}` and `onInteractOutside={(e) => e.preventDefault()}` on DialogContent. `modal` prop on Dialog.

**UI structure (shadcn/ui components):**

- Dialog with DialogContent (max-w-sm)
- DialogHeader: icon (LogIn from lucide-react, text-qualia-500) + DialogTitle "Clock In" + DialogDescription "Select the project you're working on to start your session."
- Clock-in time display row (same as checkin-modal.tsx: Clock icon + "Starting session at {format(new Date(), 'h:mm a')}")
- Label + Select (shadcn Select component) for project selection. Trigger placeholder: "Select a project...". Show loading state ("Loading projects...") when loadingProjects. Map projects to SelectItem. If no projects available after loading, show a muted info note "No projects assigned — contact your admin."
- Error display: same pattern as checkin-modal.tsx (AlertCircle + destructive border)
- Submit Button: disabled when !selectedProjectId || isPending || loadingProjects. Text: "Clock In" / "Clocking In..." with Loader2 spinner. Classes: `bg-qualia-500 text-white hover:bg-qualia-600 gap-2`

**Imports needed:** Dialog/DialogContent/DialogHeader/DialogTitle/DialogDescription from @/components/ui/dialog, Button from @/components/ui/button, Select/SelectContent/SelectItem/SelectTrigger/SelectValue from @/components/ui/select, Label from @/components/ui/label, LogIn/Clock/AlertCircle/Loader2 from lucide-react, clockIn from @/app/actions/work-sessions, invalidateActiveSession from @/lib/swr, getEmployeeAssignments from @/app/actions/project-assignments, format from date-fns, useState/useEffect/useTransition from react.
