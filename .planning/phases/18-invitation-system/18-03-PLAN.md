**1. Update getProjectsForPortalImport (app/actions/portal-import.ts):**

Add LEFT JOIN to client_invitations to get latest invitation status:

```sql
SELECT DISTINCT ON (p.id)
  p.*,
  COUNT(cp.id) as portal_access_count,
  ci.id as invitation_id,
  ci.email as invited_email,
  ci.status as invitation_status,
  ci.invited_at,
  ci.resent_count
FROM projects p
LEFT JOIN client_projects cp ON p.id = cp.project_id
LEFT JOIN client_invitations ci ON p.id = ci.project_id
WHERE p.workspace_id = $workspaceId
GROUP BY p.id, ci.id
ORDER BY p.id, ci.invited_at DESC NULLS LAST
```

Update ProjectForImport type to include:

- invitationId?: string
- invitedEmail?: string
- invitationStatus?: 'sent' | 'resent' | 'opened' | 'accepted'

function getInvitationBadge(project: ProjectForImport) {
  if (!project.invitationStatus) return null;

  const config = {
    sent: { label: 'Invited', color: 'blue', icon: Mail },
    resent: { label: 'Reminder Sent', color: 'purple', icon: RefreshCw },
    opened: { label: 'Link Opened', color: 'amber', icon: Eye },

}
