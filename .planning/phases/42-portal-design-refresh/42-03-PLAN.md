**Touch target sizes**: Any `button`, `Link`, or interactive element that is currently under 44px height/width:

- Filter/tab buttons in `portal-request-list.tsx` (already has this component imported via requests page): current `px-2.5 py-1.5` gives roughly 32px height. Add `min-h-[40px]` to the tab buttons or increase padding to `py-2.5`.

- Invoice list: if there's a table or wide row layout — ensure the container has `overflow-x-auto` with a `min-w-[400px]` inner wrapper (not min-w-full which causes overflow).
