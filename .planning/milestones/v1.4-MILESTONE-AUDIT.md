## End-to-End Flow

### Admin Import -> Client Portal Access (E2E)

```
Admin: View ERP projects (/admin/projects/import)
  -> Select project, configure portal settings (metadata.portal_settings)
  -> Send invitation email (client_invitations + Resend email)
  -> Track invitation status (badges, history modal)
Client: Receive invitation email
  -> Click signup link (/auth/signup?token={TOKEN})
  -> Token validated server-side, email pre-filled
  -> Create account (signUp() creates session)
  -> Auto-redirect to /portal/[projectId]
  -> View roadmap, files, comments immediately
```

**Flow status: COMPLETE** — All steps verified via code analysis. Manual testing recommended before production use.
